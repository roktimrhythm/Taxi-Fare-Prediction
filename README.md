# 🚖 Taxi Fare Prediction using Machine Learning

## 📌 Overview
This project predicts taxi fares based on travel distance using a Linear Regression model.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 📂 Files
- assignment3.ipynb - Model training
- app.py - Streamlit application
- taxi_model.pkl - Saved machine learning model

## ▶️ Run the Project

```bash
streamlit run app.py
```

## 🤖 Model
- Algorithm: Linear Regression
- Feature: Distance
- Target: Fare
